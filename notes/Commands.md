# Linux basic commands

- `man` displays manual of a command.  **man ls**
- `--help`  : a parameter to display help information of a command.  **ls --help**
- `who` : shows logged in users
- `whoami` : displays current user
- `date` : displays date and time
- `uname` : display system information.  **uname**  **uname -a**
- `pwd` : displays working directory.
- `cd` : copy file; requires source and destination path.   **cp myfile.txt myFiles/docs/**
- `mv` : rename/move files. **mv myFile.txt list.txt**  **mv file.txt /path/dest/**
- `mkdir` : create directory.   **mkdir myDir**
- `rmdir` : remove directory.   **rmdir myFile.txt**
- `cat` : display content of a file. **cat myFile.txt**
- `less / more` : navigate through the content of a file. **less**
- `nano` : simple text editor. **nano myFile.cp**   ctrl+o =save   ctrl+x = exit editor
- `ls` list files and directories   **ls -l**
- `chmod` : change file permission
- `ps` : list processes   **ps -f**  **ps -ef**
- `df` : display disk space usuage   **df -h**
